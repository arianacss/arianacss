Grid System

A powerful 12-column CSS Grid layout system for building responsive, structured pages. Mobile-first, semantic, and easy to learn.

Two Grids, One System
ArianaCSS has two grid systems that work together:
  • Layout Grid — The 12-column page layout system (ar-grid, ar-col-*)
  • Utility Grid — Arbitrary grid configurations (ar-grid-cols-*, ar-col-start-*)
Use the layout grid for page structure. Use the utility grid for custom grids inside sections.

Introduction

ArianaCSS uses a 12-column CSS Grid layout system. It's mobile-first, uses minmax(0, 1fr) for intelligent column sizing, and provides responsive variants at all four breakpoints.

The Container

Before using the grid, wrap your content in a .ar-container. The container provides centered, max-width-constrained content with responsive padding.

<div class="ar-container"> <!-- Your content here --> </div>

.ar-container — centered, max-width, with responsive padding

Fluid Container

Use .ar-container--fluid for full-width sections that still need consistent padding.

<div class="ar-container--fluid ar-bg-primary-surface ar-py-6"> <div class="ar-container"> <!-- Inner content still constrained --> </div> </div>

.ar-container--fluid — full width, padded inner content

Grid Basics

The grid is created with .ar-grid. By default, it creates 12 equal-width columns.

Column Spans

Use .ar-col-{n} to control how many columns an element spans. Available: ar-col-1 through ar-col-12.

Full-Width Columns

Use .ar-col-full or .ar-col-12 for full-width elements. In a standard 12-column grid, both produce the same result.

Responsive Grid

ArianaCSS is mobile-first. Use breakpoint prefixes to control column spans at different screen sizes.

Prefix Min-Width Device Class
ar-sm:col-* 480px Narrow → standard phone
ar-md:col-* 768px Phone → tablet
ar-lg:col-* 1024px Tablet → laptop
ar-xl:col-* 1280px Laptop → wide desktop

Gap Variants

Control the spacing between grid columns with gap variants.

Class Gap Size Use Case
.ar-grid--gap-sm 16px (1rem) Dense dashboards, compact layouts
.ar-grid (default) 24px (1.5rem) Standard page layouts
.ar-grid--gap-lg 32px (2rem) Editorial, generous layouts
.ar-grid--no-gap 0 Flush columns, image grids

Breakout Elements

Use .ar-breakout to make an element extend to the full viewport width while staying inside a container. Perfect for full-width images, banners, and sections.

<div class="ar-container"> <p>Content inside the container.</p> <div class="ar-breakout"> <div class="doc-breakout-demo"> This element spans the full viewport width! </div> </div> <p>Back to normal container content.</p> </div>

Content inside the container.

This element spans the full viewport width!

Back to normal container content.

The blue banner extends to the full viewport width, breaking out of the container.

⚠️ Note: .ar-breakout requires overflow-x: hidden on the html or body element (set in base.css) to prevent horizontal scroll.

Narrow Container

Use .ar-container--narrow for content that needs a tighter reading width. Perfect for blog posts, articles, and documentation.

<div class="ar-container ar-container--narrow"> <h2>The Art of Typography</h2> <p>Long-form content with a comfortable line length.</p> </div>

The Art of Typography

Typography is the visual component of the written word. A well-crafted typographic system creates rhythm, hierarchy, and harmony in any design.

The narrow container keeps lines to approximately 65 characters — the optimal reading width for long-form content.

.ar-container--narrow uses max-width: 65ch

Utility Grid

For custom grid structures inside page sections, use the utility grid system from 5-utilities/grid.css. This provides arbitrary column configurations.

<!-- Custom 3-column grid inside a section --> <div class="ar-grid-cols-3 ar-gap-6" style="display:grid;"> <div class="ar-card">Card 1</div> <div class="ar-card">Card 2</div> <div class="ar-card">Card 3</div> </div> <!-- 1-col mobile, 2-col tablet, 3-col desktop --> <div style="display:grid;" class="ar-grid-cols-1 ar-md:grid-cols-2 ar-lg:grid-cols-3 ar-gap-6"> <div class="ar-card">Card 1</div> <div class="ar-card">Card 2</div> <div class="ar-card">Card 3</div> <div class="ar-card">Card 4</div> </div>

Feature 1

Responsive grid utility

Feature 2

Arbitrary column count

Feature 3

Combines with gap utilities

Feature 4

Works with display:grid

Mobile: 1 column · Tablet: 2 columns · Desktop: 3 columns

💡 Tip: The utility grid does not set display: grid. You must apply display: grid separately via the ar-grid utility or inline styles. This allows you to use grid utilities on any element.
