Features

Everything you need to build modern UIs

A lightweight framework with a unique design language, accessible components, and utilities that work together seamlessly.

🎨

Unique Design Language

Inspired by Islamic geometric principles and Afghan natural colors — Lapis blue, Saffron gold, Cedar green.

⚡

Lightweight & Fast

Under 15 KB minified. No JavaScript dependencies. Pure CSS that loads instantly on any connection.

♿

Accessibility First

WCAG 2.2 compliant focus rings, ARIA support, reduced motion, and high contrast mode built in from the start.

📱

Mobile-First

Responsive utilities for all breakpoints. Everything works on phones, tablets, and desktops out of the box.

🧩

Component + Utility Hybrid

Pre-built components (buttons, cards, alerts) plus composable utilities for complete flexibility.

🌍

RTL Ready

Full support for Dari, Pashto, Arabic, and Hebrew with logical CSS properties throughout the framework.
