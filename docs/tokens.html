Tokens

The complete design system. Every visual decision in ArianaCSS is defined as a CSS custom property. Learn the tokens, master the framework.

Token Naming Convention
Tokens follow the pattern: --ar-[category]-[subcategory]-[variant]. See TOKEN-STANDARD.md for the complete grammar.

Introduction

ArianaCSS uses a token system with 147 tokens organized into 17 sections. The token system is the single source of truth for every visual decision in the framework.

Color Primitives

The raw palette. Five hues with five perceptually-calibrated stops each. These are the foundation of the color system.

Hue: 226° — between indigo and royal blue. Distinct from Bootstrap and Tailwind.

Semantic Colors

Semantic tokens define what colors mean. Components and utilities reference these, not the primitive tokens directly.

Primary (Interactive)

--ar-color-primary-surface var(--ar-lapis-50)
--ar-color-primary-border var(--ar-lapis-300)
--ar-color-primary-base var(--ar-lapis-600)
--ar-color-primary-hover var(--ar-lapis-700)

Typography

All typography decisions—font size, weight, line height, and letter spacing— are defined as tokens.

Type Scale

Token Value Size Use
--ar-text-size-xs 0.75rem 12px Captions, badges
--ar-text-size-sm 0.875rem 14px Form labels, secondary text
--ar-text-size-base 1rem 16px Body text (browser default)
--ar-text-size-lg 1.125rem 18px Lead text, intro paragraphs
--ar-text-size-xl 1.25rem 20px Card titles, h6
--ar-text-size-2xl 1.5rem 24px Section headings, h5
--ar-text-size-3xl 1.875rem 30px Prominent headers, h4
--ar-text-size-4xl 2.25rem 36px Page section titles, h3
--ar-text-size-5xl 3rem 48px Major headings, h2
--ar-text-size-6xl 3.75rem 60px Hero headings, h1 (desktop)

Spacing Scale

The spacing scale is based on a 4px base unit with an 8px primary rhythm. 17 steps from 0 to 96px.

Token Value Step Use
--ar-space-0 0 — Explicit zero
--ar-space-hairline 1px — Border offsets, thin separators
--ar-space-micro 2px 0.5 Ultra-fine: icon gaps, checkbox labels
--ar-space-1 4px 1 Tight internal component gaps
--ar-space-1h 6px 1.5 Badge padding, chip internal
--ar-space-2 8px 2 Button padding-y, input padding-y
--ar-space-3 12px 3 Form label gap, card header y
--ar-space-4 16px 4 Standard component padding
--ar-space-5 20px 5 Section-internal spacing
