Grid Utilities

Build custom grid layouts with flexible column configurations, item placement, and auto-flow control. Perfect for card grids, dashboards, and complex layouts.

Layout Grid vs Utility Grid
Layout Grid (docs/grid.html): The 12-column page layout system. Use for page structure.
Utility Grid (this page): Arbitrary grid configurations inside sections. Use for custom grids.

Introduction

Grid utilities provide complete control over CSS Grid layouts. Unlike the 12-column layout grid, these utilities let you create any grid configuration — 1, 2, 3, 4, 6, or 12 columns, with custom item placement and flow control.

Grid Container

Grid utilities do not set display: grid. You must use ar-grid (from display utilities) to create a grid container. Then use grid utilities to configure it.

<!-- Always start with ar-grid --> <div class="ar-grid ar-grid-cols-3 ar-gap-4"> <div>Item 1</div> <div>Item 2</div> <div>Item 3</div> </div>

Column Start & End

Use ar-col-start-{n} and ar-col-end-{n} for precise item placement in a grid.

<div class="ar-grid ar-grid-cols-4 ar-gap-4"> <div class="ar-col-start-2 ar-col-end-4">Starts at 2, ends at 4</div> <div class="ar-col-start-1 ar-col-end-5">Full row</div> </div>

📌 Note: Grid lines are 1-indexed. In a 4-column grid, lines go from 1 to 5. col-start-2 col-end-4 places the item between lines 2 and 4 (spanning columns 2 and 3).

Common Mistakes to Avoid

❌ Missing ar-grid

<div class="ar-grid-cols-3 ar-gap-4">...</div>

Why: Grid utilities require display: grid. Without ar-grid, the grid utilities have no effect.

✅ Always include ar-grid

<div class="ar-grid ar-grid-cols-3 ar-gap-4">...</div>

Why: ar-grid sets display: grid and enables all grid utilities to work correctly.

❌ Mixing layout and utility grids

<div class="ar-grid ar-grid-cols-12"> <div class="ar-col-3">Sidebar</div> <div class="ar-col-9">Content</div> </div>

Why: This creates a 12-column grid, but the layout grid (ar-grid + ar-col-*) already exists in 3-layout/grid.css. Use the layout grid for page structure.

✅ Use layout grid for page structure

<div class="ar-grid"> <div class="ar-col-3">Sidebar</div> <div class="ar-col-9">Content</div> </div>

Why: The layout grid is designed for page-level structure with 12 columns and responsive spans. Utility grid is for custom configurations inside sections.

Quick Reference

Grid Container

  • ar-grid — display: grid
  • ar-inline-grid — display: inline-grid
  • ar-grid-cols-{n} — 1, 2, 3, 4, 6, 12, auto
  • ar-grid-rows-{n} — 1, 2, 3, 4, auto

Item Placement

  • ar-col-{n} — span 1–12 columns
  • ar-col-full — span all columns
  • ar-col-auto — auto placement
  • ar-col-start-{n} — start at line 1–13
  • ar-col-end-{n} — end at line 1–13
  • ar-row-span-{n} — span 1–6 rows
  • ar-row-span-full — span all rows

Grid Flow

  • ar-grid-flow-row — row (default)
  • ar-grid-flow-col — column
  • ar-grid-flow-dense — dense packing
  • ar-grid-flow-row-dense — row dense
  • ar-grid-flow-col-dense — column dense

Auto Tracks

  • ar-auto-cols-auto/min/max/fr
  • ar-auto-rows-auto/min/max/fr

Placement

  • ar-place-center/start/end/stretch
  • ar-place-self-* — individual alignment
📚 Responsive Prefixes: sm: 480px, md: 768px, lg: 1024px, xl: 1280px
