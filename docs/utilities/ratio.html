Ratio Utilities

Maintain consistent aspect ratios for images, videos, and responsive containers. Six named ratios with responsive variants for every layout need.

Modern CSS aspect-ratio
Ratio utilities use the modern aspect-ratio CSS property. Supported in all target browsers: Chrome 88+, Firefox 89+, Safari 15+. No padding-bottom hacks required.

Introduction

Ratio utilities ensure content maintains a fixed aspect ratio regardless of container width. Perfect for images, videos, embeds, and responsive cards.

Ratio Removal

<div class="ar-ratio-video ar-lg:ratio-auto">...</div>

Use ar-ratio-auto to restore the element's natural intrinsic ratio. Perfect for responsive behavior where a fixed ratio is only needed at certain breakpoints.

Responsive Ratios

All ratio utilities have responsive variants at all breakpoints. Use them to change aspect ratio at different screen sizes.

<!-- Mobile: square · Tablet: 4:3 · Desktop: 16:9 --> <div class="ar-ratio-square ar-md:ratio-photo ar-lg:ratio-video"> <img style="object-fit:cover;width:100%;height:100%;" src="image.jpg" alt="Responsive image"> </div> <!-- Mobile: portrait · Desktop: auto (natural) --> <img class="ar-ratio-portrait ar-xl:ratio-auto" src="image.jpg" alt="Image">

Common Mistakes to Avoid

❌ Missing object-fit on images

<div class="ar-ratio-video"> <img src="image.jpg" alt="Image"> </div>

Why: Images inside ratio containers need object-fit: cover or contain to fill the ratio container properly.

✅ Always use object-fit

<div class="ar-ratio-video"> <img src="image.jpg" alt="Image" style="object-fit:cover;width:100%;height:100%;"> </div>

Why: object-fit: cover ensures the image fills the ratio container without distortion.

❌ Using ratio on the wrong element

<img class="ar-ratio-video" src="image.jpg" alt="Image">

Why: aspect-ratio works on images, but it's more commonly applied to a container that wraps the image or video.

✅ Apply ratio to containers

<div class="ar-ratio-video ar-overflow-hidden"> <img src="image.jpg" alt="Image" style="object-fit:cover;width:100%;height:100%;"> </div>

Why: Wrapping the image in a ratio container gives you more control over layout and overflow behavior.

Quick Reference

Named Ratios

  • ar-ratio-square — 1:1
  • ar-ratio-video — 16:9
  • ar-ratio-photo — 4:3
  • ar-ratio-portrait — 3:4
  • ar-ratio-wide — 21:9
  • ar-ratio-golden — 1.618:1
  • ar-ratio-auto — natural ratio

Responsive Prefixes

  • ar-sm:ratio-* — 480px
  • ar-md:ratio-* — 768px
  • ar-lg:ratio-* — 1024px
  • ar-xl:ratio-* — 1280px

Common Use Cases

  • Square: Avatars, thumbnails
  • Video: YouTube, hero banners
  • Photo: Blog images, products
  • Portrait: Mobile photos, book covers
  • Wide: Cinema banners, headers
  • Golden: Editorial, feature cards
💡 Pro Tip: Ratio utilities work best when combined with ar-overflow-hidden and ar-rounded-* for polished, production-ready image and video containers.
