Flexbox Utilities

Build flexible, responsive layouts with a complete set of flexbox utilities. Direction, alignment, distribution, and ordering — all in one system.

Important: Display First
Flexbox utilities do not set display: flex. Always use ar-flex or ar-inline-flex (from display utilities) before using flexbox utilities.

Introduction

ArianaCSS provides a complete set of flexbox utilities for building flexible layouts. Use them for navigation bars, card grids, button groups, form rows, and more.

Direction & Wrap

Control the flow direction and wrapping behavior of flex containers.

<div class="ar-flex ar-flex-row ar-gap-4"> <span>Item 1</span> <span>Item 2</span> <span>Item 3</span> </div> <div class="ar-flex ar-flex-col ar-gap-4"> <span>Item 1</span> <span>Item 2</span> <span>Item 3</span> </div> <div class="ar-flex ar-flex-wrap ar-gap-4"> <span>Item 1</span> <span>Item 2</span> <span>Item 3</span> <span>Item 4</span> </div>

Row (default)

ar-flex-row
1 2 3

Column

ar-flex-col
1 2 3

Wrap

ar-flex-wrap
1 2 3 4 5

Items wrap to the next row when space runs out

Responsive Direction

<div class="ar-flex ar-flex-col ar-md:flex-row ar-gap-4"> <span>Stack on mobile, row on tablet+</span> <span>Item 2</span> <span>Item 3</span> </div>

Flex Shorthand

Use flex shorthand utilities for common flex patterns: ar-flex-1, ar-flex-auto, ar-flex-initial, and ar-flex-none.

<div class="ar-flex ar-gap-4"> <div class="ar-flex-1">Fills remaining space</div> <div class="ar-flex-none">Fixed width</div> </div>

ar-flex-1

flex: 1 1 0% — fills remaining space
Fill Fixed

ar-flex-auto

flex: 1 1 auto — fills from natural size
Auto Auto Fixed

ar-flex-initial

flex: 0 1 auto — default behavior
Initial Initial Fixed

ar-flex-none

flex: none — no grow/shrink
None Fill None
💡 Pro Tip: ar-flex-1 is the most commonly used flex shorthand. It makes an element fill available space in a flex container. Perfect for navigation items, form fields, and card content.

Self Alignment

Use ar-self-* to override the parent's align-items for individual flex items.

<div class="ar-flex ar-items-start" style="min-height: 120px;"> <div class="ar-self-start">Top</div> <div class="ar-self-center">Center</div> <div class="ar-self-end">Bottom</div> </div>
Self Start Self Center Self End Self Stretch

Parent has ar-items-start, but each item overrides its alignment

Responsive Flex

Direction, wrap, and alignment utilities have responsive variants at all breakpoints. Use them to change flex behavior at different screen sizes.

<!-- Stack on mobile, row on tablet+ --> <div class="ar-flex ar-flex-col ar-md:flex-row ar-gap-4"> <div>Item 1</div> <div>Item 2</div> <div>Item 3</div> </div> <!-- Center on mobile, space-between on desktop --> <nav class="ar-flex ar-justify-center ar-lg:justify-between ar-gap-4"> <a href="#">Home</a> <a href="#">About</a> <a href="#">Contact</a> </nav>

Common Mistakes to Avoid

❌ Missing display:flex

<div class="ar-items-center">...</div>

Why: Flexbox utilities require display: flex to work. Without it, alignment and direction utilities have no effect.

✅ Always include ar-flex

<div class="ar-flex ar-items-center">...</div>

Why: ar-flex sets display: flex and enables all flexbox utilities to work correctly.

❌ Mixing justify and items

<div class="ar-flex ar-justify-center ar-items-center">...</div>

Why: This works but only if you understand that justify is for the main axis and items is for the cross axis. They do different things.

✅ Understand axis directions

<!-- Row: justify=horizontal, items=vertical --> <div class="ar-flex ar-flex-row ar-justify-between ar-items-center">...</div> <!-- Column: justify=vertical, items=horizontal --> <div class="ar-flex ar-flex-col ar-justify-center ar-items-stretch">...</div>

Why: The axis direction changes with flex-direction. Plan your layout accordingly.

Quick Reference

Direction & Wrap

  • ar-flex-row — row (default)
  • ar-flex-col — column
  • ar-flex-row-reverse — reverse row
  • ar-flex-col-reverse — reverse column
  • ar-flex-wrap — wrap items
  • ar-flex-nowrap — no wrap (default)

Flex Shorthand

  • ar-flex-1 — flex: 1 1 0%
  • ar-flex-auto — flex: 1 1 auto
  • ar-flex-initial — flex: 0 1 auto
  • ar-flex-none — flex: none

Alignment & Distribution

  • ar-items-start/center/end/stretch/baseline
  • ar-justify-start/center/end/between/around/evenly
  • ar-content-start/center/end/between/around/evenly
  • ar-self-auto/start/center/end/stretch

Item Behavior

  • ar-grow — flex-grow: 1
  • ar-grow-0 — flex-grow: 0
  • ar-shrink — flex-shrink: 1
  • ar-shrink-0 — flex-shrink: 0
  • ar-basis-auto/full/0 — flex-basis
  • ar-order-first/last/none — order
