Typography Utilities

Fine-grained control over font size, weight, alignment, color, and more. Build precise typographic compositions with a complete set of utilities.

Heading Classes vs Typography Utilities
Heading Classes (.ar-h1–.ar-h6): Complete heading compositions (size + weight + leading + tracking + color).
Typography Utilities (this page): Individual properties for fine-grained control.

Introduction

Typography utilities give you complete control over text styling. Use them individually or combine them for precise typographic control.

Text Alignment

ar-text-left — Left aligned text

ar-text-center — Centered text

ar-text-right — Right aligned text

ar-text-justify — Justified text that stretches to fill the full width of the container.

ar-text-start — Start aligned (left in LTR, right in RTL)

ar-text-end — End aligned (right in LTR, left in RTL)

🌍 RTL Note: ar-text-start and ar-text-end are logical properties. In RTL languages (Dari, Pashto, Arabic), start aligns to the right and end aligns to the left.

Common Mistakes to Avoid

❌ Using heading utilities on body text

<p class="ar-h3">This is body text styled as a heading</p>

Why: Heading classes (ar-h1–ar-h6) are complete compositions. They may include margin and letter-spacing that are inappropriate for body text.

✅ Use size + weight utilities

<p class="ar-text-3xl ar-font-bold"> Body text with heading size and weight </p>

Why: Combined utilities give you precise control without unintended side effects.

❌ Overusing text colors

<span class="ar-text-primary">Important info</span>

Why: Primary text color should be used sparingly. It's for the main interactive color, not for random emphasis.

✅ Use semantic colors appropriately

<!-- For emphasis, use font-weight --> <span class="ar-font-semibold">Important info</span> <!-- For status, use semantic colors --> <span class="ar-text-success">✓ Success</span> <span class="ar-text-danger">✗ Error</span>

Why: Semantic colors communicate meaning (success, error, warning). For general emphasis, use weight or other visual treatments.
