Getting Started

Learn how to install and use ArianaCSS in your project. Start building modern, accessible, and lightweight web interfaces in minutes.

Introduction

ArianaCSS is a pure CSS framework that combines the best ideas from Bootstrap and Tailwind CSS. It's designed to be lightweight (~15 KB gzipped), accessible, and easy to learn.

🎯 Component-First

Ready-to-use components with semantic class names.

🔧 Utility-Enabled

Curated utility classes for fine-grained control.

🎨 Theme Ready

CSS custom properties make theming effortless.

Installation

CDN (Recommended)

The easiest way to get started:

<!-- Full framework (components + utilities) --> <link rel="stylesheet" href="https://cdn.example.com/arianacss/dist/ariana.min.css"> <!-- Or core-only (components only, no utilities) --> <!-- <link rel="stylesheet" href="https://cdn.example.com/arianacss/dist/ariana-core.min.css"> --> <!-- Or utilities only --> <!-- <link rel="stylesheet" href="https://cdn.example.com/arianacss/dist/ariana-utilities.min.css"> -->

Package Manager

# npm npm install arianacss # yarn yarn add arianacss # pnpm pnpm add arianacss

Then import in your project:

/* Import everything */ @import 'arianacss/dist/ariana.min.css'; /* Or import core only */ @import 'arianacss/dist/ariana-core.min.css'; /* Or import utilities only */ @import 'arianacss/dist/ariana-utilities.min.css';

Direct Download

Download the latest release from the releases page and include it in your project:

<link rel="stylesheet" href="path/to/ariana.min.css">

Quick Start

Here's a minimal example to get you started:

<!DOCTYPE html> <html lang="en"> <head> <meta charset="UTF-8"> <meta name="viewport" content="width=device-width, initial-scale=1.0"> <title>My ArianaCSS Project</title> <link rel="stylesheet" href="https://cdn.example.com/arianacss/dist/ariana.min.css"> </head> <body> <div class="ar-container"> <h1>Hello, ArianaCSS!</h1> <p>This is a paragraph with <a href="#">a link</a>.</p> <button class="ar-btn ar-btn--primary">Get Started</button> </div> </body> </html>

Core Concepts: Tokens

ArianaCSS is built on a comprehensive token system. Every visual decision—color, spacing, typography, shadow—is defined as a CSS custom property.

:root { /* Primitive tokens */ --ar-lapis-600: hsl(226, 72%, 46%); /* Semantic tokens */ --ar-color-primary-base: var(--ar-lapis-600); --ar-color-primary-hover: var(--ar-lapis-700); }

Learn more: See the Tokens documentation for the complete reference.

Core Concepts: Layers

ArianaCSS uses CSS @layer to manage the cascade. The layer order is:

@layer tokens, base, layout, typography, components, utilities;

Utilities win over components. This means you can override any component style with a utility class without using !important.

Learn more: See LAYER-CONTRACT.md for the complete layer documentation.

Core Concepts: Responsive Design

ArianaCSS is mobile-first. Breakpoints:

Breakpoint Value Device Class
sm 480px Narrow phone → standard phone
md 768px Phone → tablet
lg 1024px Tablet → laptop
xl 1280px Laptop → wide desktop

Use responsive prefixes:

<!-- Hide on mobile, show as flex on tablet+ --> <nav class="ar-hidden ar-md:flex">...</nav> <!-- Stack on mobile, row on tablet --> <div class="ar-flex ar-flex-col ar-md:flex-row">...</div>

💡 Tip: All examples in the documentation use live ArianaCSS classes. You can copy and paste them directly into your project.
