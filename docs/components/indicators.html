Indicators

Two small, essential UI components — Badge and Avatar — combined for efficiency. Perfect for status labels, notifications, user profiles, and team displays.

Two Components in One
Badges and avatars share structural similarities and are often used together. They're combined in a single file for efficiency while remaining fully independent.

Badge

A small label used to communicate status, count, category, or metadata. Badges are inline elements that attach to or appear alongside other content.

💡 Badge Variants: Surface (tinted), Filled (solid), Outline (border only), and Dot (status indicator).

Accessibility

✅ Badge Best Practices

  • Use aria-label on dot badges
  • Interactive badges need tabindex="0" and role="button"
  • Badges should be <span> or <div> with proper semantics
  • Ensure sufficient color contrast (verified in token system)

✅ Avatar Best Practices

  • Images must have alt text
  • Initials avatars should have aria-label with the full name
  • Interactive avatars need tabindex="0" and role="button"
  • Avatar groups should be announced as a list when possible

Accessible Examples

<!-- Accessible dot badge --> <span class="ar-badge--dot ar-badge--success" aria-label="User is online"></span> <!-- Accessible initials avatar --> <div class="ar-avatar ar-avatar--initials ar-avatar--primary" aria-label="Jane Doe"> <span>JD</span> </div> <!-- Accessible interactive avatar --> <div class="ar-avatar ar-avatar--interactive ar-avatar--initials ar-avatar--primary" tabindex="0" role="button" aria-label="Open profile: Jane Doe"> <span>JD</span> </div>

Do / Don't

✅ Do: Use semantic variants

<span class="ar-badge ar-badge--success">Published</span> <span class="ar-badge ar-badge--warning">Draft</span> <span class="ar-badge ar-badge--danger">Archived</span>

Match the badge role to the status meaning.

❌ Don't: Use wrong color for status

<span class="ar-badge ar-badge--danger">Published</span>

Danger should only be used for errors or destructive states.

✅ Do: Use alt text for avatars

<div class="ar-avatar"> <img src="user.jpg" alt="Jane Doe"> </div>

Always include alt text for image avatars.

❌ Don't: Skip alt text

<div class="ar-avatar"> <img src="user.jpg"> </div>

Images without alt text are inaccessible to screen readers.
